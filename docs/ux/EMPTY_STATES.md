# ChileHub UX - Empty States

## Proposito

Los estados vacios deben ayudar al usuario a recuperarse. Nunca deben ser pantallas muertas ni mensajes tecnicos.

Cada empty state debe incluir:

- Que paso.
- Por que importa.
- Que puede hacer ahora.

## Tono

- Humano.
- Claro.
- Breve.
- Sin culpa.
- Sin burocracia.

## No resultados de busqueda

Titulo:

"No encontramos ese tramite"

Texto:

"Prueba con otro nombre o revisa tramites relacionados. Algunos procesos pueden aparecer con nombres distintos."

Acciones:

- Buscar de nuevo.
- Ver tramites populares.
- Explorar categorias.

No usar:

- "0 resultados".
- "Error".
- "No data".

## Sin favoritos

Titulo:

"Aun no tienes favoritos"

Texto:

"Guarda tramites importantes para encontrarlos rapido despues."

Acciones:

- Explorar tramites.
- Ver populares.

## Sin historial

Titulo:

"Todavia no hay historial"

Texto:

"Cuando revises o guardes procesos, apareceran aqui para que puedas retomarlos."

Acciones:

- Buscar un tramite.
- Ver recomendaciones.

## Sin documentos

Titulo:

"No hay documentos agregados"

Texto:

"Cuando un tramite requiera documentos, te mostraremos una lista clara para prepararlos."

Acciones:

- Ver tramite activo.
- Buscar tramite.

## Sin documentos pendientes

Titulo:

"Todo listo por ahora"

Texto:

"No tienes documentos pendientes en este paso."

Acciones:

- Continuar.
- Ver resumen.

## Sin conexion

Titulo:

"Sin conexion"

Texto:

"Puedes revisar lo que ya estaba cargado. Para actualizar informacion o guardar cambios, vuelve a conectarte."

Acciones:

- Reintentar.
- Ver contenido cargado.

## Sin tramites disponibles en categoria

Titulo:

"No hay tramites en esta categoria todavia"

Texto:

"Estamos priorizando los procesos mas solicitados. Puedes buscar por nombre o revisar otras categorias."

Acciones:

- Buscar.
- Ver todas las categorias.

## Sin oficinas encontradas

Titulo:

"No encontramos oficinas para este filtro"

Texto:

"Prueba con otra comuna o revisa el canal oficial del tramite."

Acciones:

- Cambiar comuna.
- Ver informacion general.

## Sin calculadoras

Titulo:

"No hay calculadora disponible"

Texto:

"Este proceso todavia no tiene una estimacion automatica. Si existe un costo, te mostraremos la fuente disponible."

Acciones:

- Ver requisitos.
- Ver fuente.

## Error recuperable

Titulo:

"Algo no cargo bien"

Texto:

"No se perdio tu avance. Intenta nuevamente en unos segundos."

Acciones:

- Reintentar.
- Volver al inicio.

## Preparacion lista sin siguientes pasos

Titulo:

"Preparacion lista"

Texto:

"Guardamos el resumen para que puedas revisarlo antes de continuar fuera de ChileHub."

Acciones:

- Ver resumen.
- Buscar otro tramite.
